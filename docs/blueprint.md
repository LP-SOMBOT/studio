# **App Name**: Oskar Shop

## Core Features:

- Guest Browsing & Real-time Product Catalog: Allow users to browse various game top-up packages and game accounts, viewing real-time details like title, description, thumbnail, current price, and discounted price without requiring a login.
- User Authentication & Profile Management: Secure user registration, login, and basic profile management functionalities, which are mandatory for initiating any purchase actions.
- Shopping Cart & Purchase Initiation: Enable users to add and remove items from a persistent shopping cart, view their selections, and proceed to the checkout process.
- Dynamic Checkout Flow & USSD Payment Integration: Implement game-specific checkout forms (e.g., Free Fire requiring in-game details) and integrate with local payment methods like EVCPLUS/ZAAD via USSD dialer with a post-payment confirmation step.
- Real-time Order Status & History: Provide users with real-time updates on their order statuses (pending, processing, successful, cancelled) and access to their complete purchase history.
- AI-Enhanced Promotional Content Tool: An internal admin tool leveraging generative AI to suggest and refine compelling announcement texts and event descriptions based on available product data and recent shop activity for display to users.
- Admin Panel for Shop & User Management: A comprehensive, secure administrative interface for the super admin to manage products, categories, pricing, user accounts, and global shop settings in real-time.

## Style Guidelines:

- Color Scheme: Light theme, exuding energy and trustworthiness suitable for a digital storefront. Primary color: #8526CC (a vibrant purple), for key actions and branding. Background color: #F3F1F7 (a very light, desaturated purple), providing a clean canvas. Accent color: #1945F2 (a vivid blue), used for call-to-actions and important highlights.
- Headlines and prominent text: 'Space Grotesk' (sans-serif), for a modern, tech-savvy and engaging feel. Body text and general content: 'Inter' (sans-serif), chosen for its excellent readability and neutral, objective appearance, suitable for diverse product information.
- Use a consistent set of modern, clear line icons that are easily recognizable and relevant to gaming actions (e.g., cart, profile, game categories). Focus on simplicity and quick comprehension for navigation.
- Employ a mobile-first responsive design, ensuring optimal experience on smaller screens. Feature prominent image sliders and a continuously looping announcement ticker on the homepage. Organize products in clean grids with clear pricing and 'add to cart' indicators. Utilize a fixed bottom mobile navigation bar for easy access to core app sections.
- Incorporate subtle, smooth transitions for page navigation and content loading to enhance user experience. Provide clear visual feedback with quick, satisfying animations for actions like 'add to cart' and real-time order status updates. Ensure image sliders transition seamlessly.
